# korfixdev/docs

Публичная документация платформы Korfix — источник правды для всех потребителей:
- разработчиков миниапов (читают на [docs.korfix.ru](https://docs.korfix.ru))
- AI-агентов в Claude Code plugin [korfixdev/devkit](https://github.com/korfixdev/devkit) (подтягивает этот репо)
- автоматической сборки в статический сайт

## Навигация

См. [INDEX.md](INDEX.md).

## Разделы

| Папка | Аудитория | Содержание |
|-------|-----------|-----------|
| `miniapps/` | Разработчики маркетплейс-миниапов | SDK: config.json, JS API, data API, стилизация, деплой |

## Контрибьюция

1. Fork → правка → PR. Новый файл — обязательно добавь ссылку в `miniapps/INDEX.md` и блок «См. также» в шапку.
2. Каждый документ содержит вверху блок `> **См. также:**` с 2-4 ссылками на смежные темы и `> **← [INDEX](INDEX.md)**`. Внизу — `**Дальше:** X · **← [INDEX](INDEX.md)**` для линейного чтения, или только `← INDEX` для справочников.
3. Примеры кода — рабочие, должны запускаться копипастой.
